# CONVOYPLAN — Architecture

> Defense logistics route/sustainment planner computing fuel, resupply windows, and chokepoint risk from a YAML plan.

```
input ──▶ collect ──▶ rules/analyzers ──▶ score ──▶ findings ──▶ table · json
                              │                          │
                         (this repo)                 MCP tool (agents)
```

- **collect** normalizes the target (file/dir/API) into records.
- **rules/analyzers** apply the heuristics shipped in `convoyplan/core.py`.
- **score** ranks by severity.
- **MCP server** (`convoyplan mcp`) exposes `scan` for Cognis.Studio agents.

Extend by adding a rule + a test + a `demos/NN-*/SCENARIO.md`. See [CONTRIBUTING.md](../CONTRIBUTING.md).
